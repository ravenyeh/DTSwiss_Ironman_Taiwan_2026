## 💪 如何達成目標？

### 🏊 游泳策略：技術突破

- **目標**：2:49/100m → 2:30/100m（節省11分鐘）
- **方法**：
    - 每週2次技術課，提升劃頻至70-75 SPM
    - 配速間歇訓練：6 x 400m @ 2:32/100m
    - 每2週1次開放水域，適應澎湖海況

### 🚴 自行車策略：功率提升

- **目標**：26.9 km/h → 30.0 km/h（節省41分鐘）
- **方法**：
    - 每週2次 Sweet Spot 訓練，提升FTP 10-15W
    - 每週1次長距離160-180km，練習補給
    - 側風訓練，適應澎湖賽道特性
    - 磚式訓練強化騎轉跑能力

### 🏃 跑步策略：配速穩定

- **目標**：6:09/km → 5:49/km（節省15分鐘）
- **方法**：
    - 配速分層：輕鬆跑60% + 馬拉松配速25% + 節奏跑15%
    - 每週1次長跑25-28km，建立耐力
    - 磚式訓練適應騎後跑步
    - 中午時段訓練，適應澎湖高溫

### 🎯 澎湖賽道特性應對

**游泳**：開放水域適應，練習浪中維持節奏

**自行車**：平坦但風大，強化側風騎乘和低風阻姿勢

**跑步**：高溫挑戰，熱適應訓練 + 補給站必停策略
